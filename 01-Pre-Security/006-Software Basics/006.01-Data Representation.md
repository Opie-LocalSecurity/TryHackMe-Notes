# Data Representation

## 1. Color Representation: The RGB Model

Computers represent colors by mixing three primary light sources: **Red, Green, and Blue (RGB)**.

### From 8 to 16 Million Colors

- **Simple 3-Bit Model:** If each color is either "on" (1) or "off" (0), there are $2^3 = 8$ possible colors (e.g., 100 is Red, 111 is White).
- **Standard 24-Bit Model:** To achieve realism, each primary color is given **256 levels of intensity** (0–255).
    - This requires **8 bits (1 byte)** per color.
    - Total combinations: $256 \times 256 \times 256 = 16,777,216$ colors.
    - **Data Structure:** A single color is represented by **3 bytes (24 bits)**.

## 2. Number Systems and Bases

Digital systems use different bases to represent data depending on the context (physical hardware vs. human-readable code).

### Key Terms

- **Bit:** A single binary digit (0 or 1).
- **Byte (Octet):** A group of 8 bits.
- **Hexadecimal:** A base-16 system used to condense long binary strings.

### Comparison Table

| **System** | **Base** | **Digits Used** | **Typical Use Case** |
| --- | --- | --- | --- |
| **Decimal** | 10 | 0–9 | Human counting/math |
| **Binary** | 2 | 0, 1 | Low-level hardware (voltage/logic) |
| **Octal** | 8 | 0–7 | Groups of 3 bits |
| **Hexadecimal** | 16 | 0–9, A–F | Color codes, memory addresses |

## 3. Mathematical Conversions

All positional number systems follow the same logic: the value is the sum of each digit multiplied by the base raised to the power of its position (starting from 0 on the right).

### Binary to Decimal (Base-2)

To convert `1001`:

$$(1 \times 2^3) + (0 \times 2^2) + (0 \times 2^1) + (1 \times 2^0) = 8 + 0 + 0 + 1 = 9$$

### Hexadecimal to Decimal (Base-16)

Hexadecimal uses letters **A–F** to represent values **10–15**. Each Hex digit represents exactly **4 bits**.

- **Example:** `A3` in Hex.
    - `A` = 10, `3` = 3.
    - Result: $(10 \times 16^1) + (3 \times 16^0) = 160 + 3 = 163$.

### Octal to Decimal (Base-8)

Each Octal digit represents exactly **3 bits**.

- **Example:** `357` in Octal.
    - Result: $(3 \times 8^2) + (5 \times 8^1) + (7 \times 8^0) = 192 + 40 + 7 = 239$.
