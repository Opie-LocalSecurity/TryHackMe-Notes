# Become a Hacker

## Core Concepts & Terminology

Offensive security is a proactive approach to cybersecurity where professionals simulate attacks to identify and fix weaknesses before malicious actors exploit them.

- **Vulnerability:** A flaw or weakness in a system's design, implementation, or configuration.
- **Exploit:** The specific method or technique used to take advantage of a vulnerability.
- **Scope:** The strictly defined boundaries (systems, applications, and permitted actions) of a security assessment.
- **Red Teaming:** A comprehensive, authorized simulation of a real-world adversary to test defensive effectiveness.
- **Penetration Test:** A structured assessment focused on identifying and exploiting vulnerabilities to evaluate risk.

## Methodologies: The "Attacker Mindset"

Ethical hackers do not just check if a system "works"; they investigate how it can be **misused**.

1. **Information Gathering:** Identifying hidden entry points (e.g., unlinked directories like `/admin`).
2. **Chaining Weaknesses:** Linking multiple minor flaws (e.g., a hidden login page + a weak password) to create a high-impact security breach.
3. **Credential Harvesting:** Targeting usernames and passwords to gain "Authenticated Access," which unlocks sensitive data and administrative functions.

## Tools and Automation

While manual testing is possible for small tasks, automation is essential for scaling security assessments.

| **Tool** | **Purpose** | **Primary Functionality** |
| --- | --- | --- |
| **Gobuster** | Content Discovery | Automates the discovery of hidden files and directories using a **wordlist** (Brute-forcing/Enumeration). |
| **Hydra** | Credential Testing | Performs **Dictionary Attacks** by automating login attempts against a target using lists of usernames and passwords. |

### Example Command Syntax:

- **Directory Scanning:** `gobuster dir --url [URL] -w [WORDLIST]`
- **Password Cracking:** `hydra -l [USER] -P [PASS_LIST] [TARGET] [SERVICE] [MODULE_ARGS]`

## Key Outcomes

- **Risk Mitigation:** Identifying "dominoes" (vulnerabilities) before they are knocked over by attackers.
- **Security Posture Improvement:** Using assessment results to close gaps in defenses and strengthen security controls.
- **Authorized Testing:** Ensuring all activities are legal, controlled, and performed with explicit permission.
