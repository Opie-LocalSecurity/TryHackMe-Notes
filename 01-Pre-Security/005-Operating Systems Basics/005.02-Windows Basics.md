# Windows Basics

## 1. Evolution and Architecture

- **Historical Context:** Windows evolved from **MS-DOS** (CLI-based) to **Windows 1.0** (first GUI in 1985), eventually maturing from a graphical shell into a standalone operating system.
- **Modern Workspace:** * **Desktop:** The primary workspace for file/shortcut management.
    - **Taskbar:** The central control strip containing the Start Menu, Search, Task View, and System Tray (Network/Audio/Clock).

## 2. Authentication and User Access

Windows uses identity verification to determine system permissions.

| Account Type | Permissions & Scope |
| --- | --- |
| **Guest** | Highly restricted; temporary access; no system changes. |
| **Standard** | Everyday tasks; personal settings; no system-wide changes. |
| **Administrator** | Full control; software installation; user/system management.  |

## 3. File Management and System Info

- **File Explorer:** The primary tool for navigating the **hierarchical folder structure**.
- **System Information:** Accessed via `Settings > About your PC`. Provides critical data on hardware specifications, OS version, and security status.
- **Core Tools:** Built-in utilities like **Notepad** (text editing) and **Search** (indexing files/apps) facilitate daily operations.

## 4. Software Lifecycle Management

- **Installation:** Methods include the **Microsoft Store** (curated/safe) or standalone installers (** .exe** or **.msi** files) from trusted vendors.
- **Updates:** **Windows Update** handles the OS and native apps. Third-party software often requires manual updates or proprietary background updaters.
- **Uninstallation:** Managed via `Settings > Apps`, the legacy `Control Panel`, or an application's specific uninstaller.

## 5. System Configuration Tools

Windows maintains two interfaces for system management:

- **Windows Settings:** The modern, centralized UI for personalization, security, and device configuration.
- **Control Panel:** The legacy interface used for advanced administrative tasks and older system tools.

## 6. Monitoring and Performance

The **Task Manager** provides real-time visibility into system resources via five key tabs:

1. **Processes:** Active apps and their resource impact.
2. **Performance:** Real-time graphs for CPU, RAM, and Network.
3. **Users:** Resource consumption per logged-in session.
4. **Details:** Advanced process information, including **PIDs (Process IDs)**.
5. **Services:** Status of background Windows services (Running/Stopped).

## 7. Native Security Framework

Windows utilizes a multi-layered defense strategy through the **Windows Security** dashboard:

- **Virus & Threat Protection:** Real-time malware scanning.
- **Firewall & Network Protection:** Managed by **Windows Defender Firewall**, which filters traffic based on three profiles:
    - **Domain:** Organizational networks.
    - **Private:** Trusted home/lab networks.
    - **Public:** Untrusted Wi-Fi hotspots.
- **App & Browser Control:** Protection against malicious web content and unsafe files.
- **Device Security:** Hardware-level protection.
