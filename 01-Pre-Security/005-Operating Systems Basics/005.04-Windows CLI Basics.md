# Windows CLI Basics

## 1. Filesystem Navigation & Exploration

Navigating the Windows directory structure is the foundation of command-line operations.

- **`cd` (Change Directory):**
    - **View Location:** Type `cd` alone to print the current working directory path.
    - **Move Forward:** `cd <folder_name>` moves you into a specific subfolder.
    - **Move Backward:** `cd ..` moves you up one level in the directory tree.
- **`dir` (Directory Listing):**
    - **Standard List:** Displays visible files and folders in the current location.
    - **View Hidden Items:** `dir /a` reveals files and folders flagged as "hidden" by the OS.
    - **Recursive Search:** `dir /s <filename>` searches the current directory and all subdirectories for a specific file.

## 2. File Manipulation

Once a file is located, you can verify and inspect it without leaving the terminal.

- **Locating:** Use the path provided by `dir /s` to navigate to the target folder.
- **Reading Content:** Use the **`type <filename>`** command to display the text of a file directly within the terminal window.

## 3. System Information Gathering

These commands are used to identify user context, hardware identity, and network status.

| **Command** | **Purpose** | **Key Data Points to Note** |
| --- | --- | --- |
| **`whoami`** | Identifies the current user. | Username and privilege context. |
| **`hostname`** | Identifies the machine. | The network name of the computer. |
| **`systeminfo`** | Comprehensive OS/Hardware summary. | OS Name, OS Version, and System Type (x86 vs. x64). |
| **`ipconfig`** | Network configuration. | **IPv4 Address** (local ID) and **Default Gateway** (router). |
